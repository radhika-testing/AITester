#**Test Case Generator - Powered by Groq & KIMI**
-------------------------------------------------------------------------------------------
A web-based application that generates comprehensive test cases using Groq's AI API. Simply describe your feature, and the AI will generate detailed test cases including positive scenarios, negative scenarios, and edge cases.

###**Installation of kimicli**
-------------------------------------------------------------------------------------------------------
Kimi Code CLI is an AI agent that runs in the terminal, helping you complete software development tasks and terminal operations. 

Once the subscription is taken, Run the below commands-
     git clone https://github.com/MoonshotAI/kimi-cli.git
     cd kimi-cli
     Run the Kimi code
     /login
     Uv run kimi

 #### How to use-
 ------------------------------------------------------------------------------------------------------------
  1.	Open your browser: Navigate to http://localhost:5000 (kimi will give you this link)
  
  2.	Enter your Groq API Key in the API Key field (starts with gsk_...)
  
  3.	Enter a feature description or query in the text area
  
  4.	Click "Generate Test Cases"
  
  5.	Copy the generated test cases to your clipboard
